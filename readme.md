# Основные команды git
#### git init - инициализация
#### git branch -M main - создать ветку без проверок
#### git add <file_name> - добавление файлов в репозиторий
#### git commit -m "" - применение изменений
#### git remove git@github.com:user_name/repository_name - добавление удаленного репозитория по SSH-ключу
#### git push origin main - отправка изменений в репозиторий

#### git clone git@github.com:user_name/repository_name - клонирование удаленного репозитория по SSH-ключу
#### git merge origin develop - применение изменений из ветки develop
#### git fetch - просмотр изменений в удаленном репозитории
#### git pull origin develop - получение изменений из ветки develop
#### git status - просмотр изменений в локальной ветке
